##1
Car Insurance (ASP.NET MVC)


This project is a simple car insurance application built using ASP.NET MVC. It allows users to input their personal information and car details to compute a personalised insurance quote. 
The application calculates the insurance premium based on factors such as the user's age, car make and model, and driving history. 
The project demonstrates the use of MVC architecture, form handling, and basic data validation in ASP.NET MVC.

##features
user input form for personal and car details
automatic insurance quote computation based on user input
-Age
-Car make and model
-Driving history
-DUI history
-Number of speeding tickets
-Types of coverage (full coverage or liability only)

##Technologies Used
-C#
-ASP.NET MVC
SQL Server (for data storage)

##2
Newsletter Sign-Up (ASP.NET MVC)


This project is a simple newsletter sign-up application built using ASP.NET MVC. It allows users to subscribe to a newsletter by providing their email address.
The application validates the email address and stores it in a database for future use.

##features
-user input form for email address
-email validation to ensure proper format
-database storage of email addresses for future use
-confirmation message upon successful subscription
-error page for invalid email addresses or null entries
-ability to view all subscribed email addresses (admin feature)
-option to unsubscribe from the newsletter (admin feature)
-MVC architecture for separation of concerns and maintainability
-use of Entity Framework for database interactions(database-first or code-first approach)

##Architecture

Controllers;-
HomeController: Handles the main page and newsletter sign-up logic.
AdminController: Handles administrative tasks such as viewing subscribed email addresses and managing subscriptions.

Models

-signup model: Represents the data structure for newsletter sign-up, including properties for email address and validation attributes.
-NewsletterEntity1 model: Represents the database entity for storing subscribed email addresses, including properties for email and subscription date.

views

-user input form for newsletter sign-up (Home/Index.cshtml)
-Admin list view to display all subscribed email addresses (Admin/Index.cshtml)

##Technologies Used
-C#
-ASP.NET MVC
-Entity Framework (for database interactions)
-SQL Server (for data storage)
-Razor view engine (for rendering views)
-Bootstrap (for styling the user interface)
